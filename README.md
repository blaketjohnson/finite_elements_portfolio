# Finite Element Portfolio

[![Python](https://img.shields.io/badge/python-3.9%2B-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A lightweight finite element analysis (FEA) portfolio built from scratch in Python.  
This repository demonstrates **fundamental FEM skills** applied to both **heat conduction** and **structural mechanics**, based directly on coursework and exam derivations (T3, Q4, T6 elements).

---

## âœ¨ Features
- T3, Q4, **T6** shape functions; isoparametric mapping & Jacobians
- Plane stress/strain and scalar conduction constitutive laws
- Element stiffness for conduction & structural problems
- Global assembly + Dirichlet BC helper
- **Examples with saved figures** (in `docs/`): Q4/T3 heat plates, Q4 cantilever
- **Post-processing:** nodal von Mises (averaged from element centers)
- **Rigor:** Q4 patch test + **convergence study** (cantilever tip deflection)
